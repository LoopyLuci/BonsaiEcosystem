# MLP (Minimum Loveable Product)

The MLP smoke test verifies six core capabilities:

1. **Chat** — model responds to a simple prompt
2. **Code generation** — `code-writer` agent produces a file action
3. **Sandbox execution** — Python code runs and returns stdout
4. **Session persistence** — context survives across messages
5. **Feature flags** — experimental features default OFF
6. **GPU stats** — hardware stats endpoint responds
