# modules/archive — Legacy and Experimental Code

This directory contains contributed, experimental, or superseded modules that are not part of the core 102-phase implementation.

## Contents

- **python_contributed.ti** — Sentiment analysis module converted from Python via Lingua framework (Phase 3 experimental). Not actively used in current platform. Preserved for reference/historical purposes.

## Note

Code in this directory is not part of the main Omnisystem platform verification suite. It represents:
- Experimental features from earlier phases
- Contributed modules from community
- Code that has been superseded by phase-specific implementations

For active platform modules, see:
- `titan/` — Core compiler and runtime (Titan language)
- `aether/` — Distributed actor framework
- `sylva/` — Interactive REPL and UI language
- `axiom/` — Formal verification and proof system
