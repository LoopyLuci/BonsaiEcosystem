## 🎮 How to Use

### Play a Game

```bash
cd bonsai-omnisystem-languages

# Play Sylva Pong interactively
make run-sylva

# Run Aether auto-play (100 frames)
make run-aether

# Verify Axiom proofs
make run-axiom

# Compile Titan to WebAssembly
make run-titan
```

### Run Tests

```bash
# Test all 4 languages
make test

# Run sandbox harness
python3 sandbox/sandbox.py

# Windows: Run comprehensive test
.\run_omnisystem_pong.ps1 -Validate -Benchmark
```

### Integrate with Polyglot Pong

```bash
cd polyglot-pong

# Run 4-language test matrix
cargo run --release --bin polyglot-pong-orchestrator -- \
  --manifest ../languages.yaml \
  --languages Titan,Sylva,Aether,Axiom \
  --nodes 1 --rounds 1

# Watch dashboard
cargo run --release --bin polyglot-pong-dashboard
# Open http://localhost:8080
```
