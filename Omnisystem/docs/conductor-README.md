## 🚀 Quick Start

### Installation

```bash
# Clone repository
git clone <repo-url>
cd Conductor

# Build entire platform
cargo build --release --all

# Run all tests
cargo test --lib --all

# Run specific layer
cargo build --release -p conductor-core
```

### Basic Usage

```rust
use conductor::Conductor;

#[tokio::main]
async fn main() {
    // Initialize Conductor
    let conductor = Conductor::new().await.unwrap();
    
    // Use any layer
    conductor.execute_operation("list_containers").await.unwrap();
}
```

### API Access

```bash
# REST API
curl http://localhost:8080/api/containers

# GraphQL
curl -X POST http://localhost:8080/graphql \
  -H "Content-Type: application/json" \
  -d '{"query": "{ containers { id name } }"}'

# WebSocket (real-time)
ws://localhost:8080/ws

# gRPC
grpcurl -plaintext localhost:50051 describe
```
