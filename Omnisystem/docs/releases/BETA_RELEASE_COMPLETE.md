### Languages (4 Total)

|----------|--------|-----------|

- **UniIR v0.2** — Typed SSA intermediate representation (effects, regions, modalities)
- **OmniCore Kernel** — 5 subsystems (CapTable, Telemetry, Loader, Scheduler, Interpreter)
- **Distributed Runtime** — Aether actors with DHT registry and CRDT sync
- **IDE Integration** — LSP server (VS Code, Neovim, Helix, Emacs) with cross-language support
- **Universal Converter** — Omni Lingua (C→Titan certified, Python→Sylva high-fidelity)
