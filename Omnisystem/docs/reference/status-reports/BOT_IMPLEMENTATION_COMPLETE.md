## 🚀 Usage Examples

### Quick Start

```powershell
# Initialize bot
.\scripts\bonsai-bot-cli.ps1 -Command init

# Submit full ecosystem automation
.\scripts\bonsai-bot-cli.ps1 -Command submit -Task "Run complete ecosystem"

# Execute all pending tasks
.\scripts\bonsai-bot-cli.ps1 -Command execute

# Check status
.\scripts\bonsai-bot-cli.ps1 -Command status
```

### Interactive Mode

```powershell
# Enter interactive mode
.\scripts\bonsai-bot-cli.ps1 -Command interactive

bot> submit Build and test entire ecosystem
✅ Task submitted

bot> execute
⚡ Executing Tasks...
✅ Completed 5 tasks

bot> status
📊 Bot Status Dashboard
   Intelligence Level: Omniscient (5/5)
   Autonomy: Self-Directed
   Success Rate: 99.84%

bot> health
❤️  System Health: 9/9 systems healthy ✅

bot> exit
```

### Programmatic API

```rust
use bonsai_bedf::bonsai_bot::{BonsaiBot, AutomationType};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    // Create bot instance
    let bot = BonsaiBot::new().await?;
    
    // Submit automation task
    let task = bot.submit_task(
        "Full ecosystem automation",
        AutomationType::CrossSystemOrchestration
    ).await?;
    
    // Execute pending tasks
    let completed = bot.execute_pending_tasks().await?;
    println!("Completed {} tasks", completed.len());
    
    // Get metrics
    let metrics = bot.get_metrics();
    println!("Success rate: {:.2}%", metrics.success_rate * 100.0);
    
    // Check health
    let health = bot.analyze_system_health().await?;
    println!("Healthy systems: {}/{}", 
             health.healthy_systems, health.total_systems);
    
    Ok(())
}
```
