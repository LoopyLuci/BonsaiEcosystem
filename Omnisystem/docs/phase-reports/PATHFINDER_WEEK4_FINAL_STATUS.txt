WHAT EACH API ENDPOINT DOES
================================================================================

USER SERVICE (8001):
  POST   /v1/auth/login → Get JWT token
  POST   /v1/auth/register → Create account
  GET    /v1/users/me → Get profile
  PUT    /v1/users/me → Update profile
  DELETE /v1/users/me → Delete account

CONTENT SERVICE (8002):
  GET    /v1/skills → List all skills
  GET    /v1/skills/:id → Get skill + prerequisites
  GET    /v1/skills/:id/exercises → List exercises
  GET    /v1/lessons/:id → Get lesson + exercises
  GET    /v1/curriculum-paths → List curricula
  GET    /v1/search → Search skills & exercises

PERSONALIZATION SERVICE (8003):
  POST   /v1/learners/:id/exercises/:id/attempt → Submit answer
    INPUT: { exercise_id, skill_id, was_correct, response, response_time }
    OUTPUT: { skill_state, next_review_at, is_mastered, feedback }
    MAGIC HAPPENS: BKT updates P(Know), HLR calculates next review

  GET    /v1/learners/:id/skills → All skill states
  GET    /v1/learners/:id/next-skills → Skills to review (spaced rep)
  GET    /v1/learners/:id/progress → Overall metrics

PROGRESS SERVICE (8004):
  GET    /v1/learners/:id/daily-metrics → Today's stats
  GET    /v1/learners/:id/monthly-metrics → Month's stats
  GET    /v1/learners/:id/skills/:id/learning-curve → Visualization data
  GET    /v1/analytics/cohort/:id/metrics → Class analytics
  GET    /v1/analytics/skills/:id/exercises → Exercise stats
