# Archived Legacy Verify Scripts

These verification scripts are from the pre-self-hosting era (phases 10-124,
omnisystem_complete, polyglot, aion, etc.). They reference the old Rust bootstrap
binary at `titan-bootstrap/target/release/titan-bootstrap.exe`, which no longer
exists. They cannot be run against the current native Titan compiler.

The modules they tested are either:
- Superseded by the 9 official tracks (AX, C, F, H, L)
- Implemented under different paths in the current architecture
- From experimental phases that were merged into the main tracks

These scripts are preserved for historical reference only. Do not run them.
