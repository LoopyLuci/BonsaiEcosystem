## DEPLOYMENT ARCHITECTURE

### Self-Hosted (Organization)
```
├─ Single server (small) or Kubernetes (large)
├─ PostgreSQL (content DB)
├─ S3-compatible storage (media)
├─ Redis (cache)
├─ Elasticsearch (search)
└─ CDN (media delivery)
```

### Federated (Decentralized)
```
├─ Multiple independent servers
├─ ActivityPub protocol (compatible)
├─ Can interoperate with Mastodon, PeerTube, etc.
├─ Users choose which server
└─ No single point of control
```
