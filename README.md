## 🚀 Quick Start

### Build
```bash
cd Omnisystem
cargo build --release --all
```

### Test
```bash
cargo test --all --release
```

### Deploy to Kubernetes
```bash
cd infrastructure/helm
helm install omnisystem ./omnisystem/
```

### Deploy with Terraform
```bash
cd infrastructure/terraform
terraform apply
```

### Run Verification
```bash
./scripts/RUN_OMNISYSTEM_VERIFICATION.sh
```
